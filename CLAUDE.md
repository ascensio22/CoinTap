# CoinTap — Telegram Mini App

## Design System
Always read DESIGN.md before making any visual or UI decisions.
All font choices, colors, spacing, and aesthetic direction are defined there.
Do not deviate without explicit user approval.

## Project Overview
- **Project:** CoinTap — Telegram Mini App idle clicker game
- **Stack:** HTML5 + CSS + Vanilla JS (no framework)
- **Hosting:** GitHub Pages (validation phase)
- **Telegram Integration:** WebApp API + Cloud Storage

## Quick Start
1. `index.html` + `style.css` + `game.js` — core files
2. Telegram WebApp SDK — for Telegram integration
3. Cloudflare Pages / GitHub Pages — for hosting
